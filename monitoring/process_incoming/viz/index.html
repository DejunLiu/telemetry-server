Telemetry: Process Incoming

This is a set of graphs showing the error rate in the code that processes raw incoming telemetry submissions.

Overall Error Rate

This is the overall percentage of incoming submissions that do not pass validation. Some are expected (such as submissions from old Firefox versions), but others may signal a problem.

Interesting Error Rate

This is the error rate excluding known error types.

Linked Charts

It can be useful to have two charts linked together—when a rollover in one is triggered, the same rollover is triggered on the same date or data point in the other.

Update Chart Data

We sometimes have the need to split the data and then gracefully update the chart with the newly selected subset of data.

Modify Time Period

We sometimes have the need to view data for just the past n days. Here, the transition_on_update option is set to false.
